Electrical Load Monitoring & Billing System

A C++ console-based system that simulates electrical energy monitoring and billing.

Features

Appliance registration

Daily and monthly energy calculation

Electricity billing simulation

Remove appliance feature

Data persistence using file handling

Concepts Used

Object-Oriented Programming

Arrays

File Handling

Input validation

Menu-driven interface

How to Compile
g++ main.cpp -o load_monitor
.\load_monitor
