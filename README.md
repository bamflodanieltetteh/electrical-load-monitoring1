Electrical Load Monitoring & Billing System (C++)
Overview

This project is a console-based C++ application that simulates electrical energy monitoring and billing.

Features

Appliance registration

Daily energy calculation (kWh)

Monthly energy calculation

Electricity bill estimation

Remove appliance functionality

Persistent data storage using file handling

Concepts Used

Object-Oriented Programming

Arrays

File Handling (fstream)

Input validation

Menu-driven interface

Energy Formula

Energy (kWh) = (Power × Hours) / 1000

Monthly Cost = Monthly Energy × Tariff

How to Compile
g++ main.cpp -o load_monitor
.\load_monitor
